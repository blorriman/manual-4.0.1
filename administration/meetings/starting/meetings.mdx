# Create a Meeting

**LORROS** has made it easy to start a Meeting.

1. The default way of **Starting a Meeting** is from the **Group Home Page**.

<center>
	<img src={useBaseUrl('/img/administration/groupHome-1b.jpg')} />
</center>

2. If your Group has been [**Referred**](../../groups/tabs#referred-meetings) a topic from another Group.
3. If your Group has [**Deferred**](../../groups/tabs#deferrals) an Agenda Item from a previous Meeting.

## Title of the Meeting

The **Title** of the Meeting can be used to set the **purpose of the Meeting**. Try to be as specific as possible so that it will make sense to the Members of the Group and it will be easy to find in the [**Archives**](../../../docs/started/groupHome#archives).

:::success Tip
Instead of "**Monthly Board Meeting**", use "**December 2021 Board Meeting**".
:::

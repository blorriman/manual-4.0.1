# Common Actions

**LORROS** is easy to use once you learn the basics. The following actions are common across all **LORROS** Activities.

## Adding Comments

Across **LORROS**, anytime you are allowed to add **Comments**, you will see the Add Comments tab. When you have added your Comments, you can also [add Files](#add-files-to-submissions) before submitting your Comment.

<center>
	<img src={useBaseUrl('/img/global/addingComments-2.gif')} />
</center>

## Adding Replies

When Replying to a Comment, you will see the Reply To This Comment tab. When you have added your Reply, you can also [add Files](#add-files-to-submissions) before ubmitting your Reply.

<center>
	<img src={useBaseUrl('/img/global/addingReplies-2.gif')} />
</center>

Most submissions _(Comments, Replies, etc.)_ allow you to add files. Simply click the **Add File** tab and follow the instructions.

## Opening Files

All **Files** for each Activity _(Discussion, Meeting, etc.)_ can be accessed from the Files tab at the top. The **Files** for each Comment _(Agenda, etc.)_ can be accessed from the Files icon _(paperclip)_.

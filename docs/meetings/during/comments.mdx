# Comments & Replies

## Adding Comments

**Comments** should be relative to the **Agenda Item** _(**Replies** should be specific to a **Comment**)_.

## Anonymous Comments

The Group Admin can set Comments to **Anonymous** in the event Group Members are uncomfortable having their name displayed. If Comments are set to Anonymous, all Comments are Anonymous for that Meeting.

When Anonymous Comments has been activated, an icon appears beside the Comments heading.

<center>
	<img src={useBaseUrl('/img/global/anonymous-1.gif')} />
</center>

# Draft Agenda

If the Meeting is scheduled to start in the future, the Draft Agenda is available to all Members of the Group.

## Adding Agenda Items

If the **Meeting** hasn't started, any Member can add Agenda Items to the Draft Agenda.

:::success Info
Agenda items can only be added before the Meeting starts. Once the Meeting starts the Agenda is fixed.
:::

:::success Info
Agenda items can be edited and/or deleted by the Group Admin or the Member who created the Agenda item before the Meeting starts.
:::

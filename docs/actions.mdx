# Actions

**Actions** are items that are assigned to a Group Member and given a target date. All Members of the Group can see the Action and add Comments/Replies to help move tha Action along.

:::success Info
**Notices** - You will recieve an email from **LORROS** when an **Action** starts and when it is completed _(you cannot mute Notices)_.

**Messages** - You will recieve an email from **LORROS** everytime a Member makes a Comment or Reply _(you can [mute Messages](#mute-emails))_
:::

## Starting an Action

An **Action** can be started by any Group Member.

<center>
	<img src={useBaseUrl('/img/global/addAction.gif')} />
</center>

## Reporting Progress

The Group Member **asigned** the Action, or any other Group Member can add Comments/Replies at any time that the Action is in progress.

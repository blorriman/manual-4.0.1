Simply click the **Add File** tab and follow the instructions.

:::note Note
When the **File** is uploaded you can edit _(rename)_ it or delete it.
:::

:::warning Attention
Try to use easily identifiable file names so they can be easily found. For instance, "Dec. 2021 Financial Statement" is better than "Financial Statement".
:::

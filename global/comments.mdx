## Anonymous Comments

The Group Admin can set Comments to **Anonymous** in the event Group Members are uncomfortable having their name displayed. If Comments are set to Anonymous, all Comments are Anonymous for that Discussion or Meeting.

:::note Note
When Anonymous Comments has been activated, an icon appears beside the Comments heading.
:::

<center>
	<img src={useBaseUrl('/img/global/anonymous-1.gif')} />
</center>

## Add Files to Submissions

Most submissions _(Comments, Replies, etc.)_ allow you to add files. Simply click the **Add File** tab and follow the instructions.

<center>
	<img src={useBaseUrl('/img/global/addFiles.gif')} />
</center>

The **Files** are accessible from both the Files tab at the top of the Discussion as well as the Paper Clip icon next to the Comment.
